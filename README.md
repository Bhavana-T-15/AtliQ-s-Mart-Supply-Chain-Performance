# AtliQ-s-Mart-Supply-Chain-Performance
Power BI dashboard analyzing supply chain delivery performance using KPIs such as OTIF, OT, IF, LIFR, and VOFR. Built interactive visuals, delay analysis, and KPI trend charts. Implemented bookmarks, buttons, DAX measures, and conditional formatting to create a dynamic and user-friendly dashboard.

# 📊 Supply Chain Performance Dashboard (Power BI)

This project is a Power BI dashboard built to analyze supply chain performance and delivery efficiency. The dashboard helps stakeholders monitor key metrics related to order fulfillment, delivery delays, and service level performance.

# 🎯 Objective

The goal of this dashboard is to provide insights into how well orders are delivered on time and in full, and to identify areas where supply chain performance can be improved.

# 📈 Key Metrics Analyzed

The dashboard focuses on the following KPIs:

OTIF % (On Time In Full) – Percentage of orders delivered on time and in full

OT % (On Time Delivery) – Percentage of orders delivered on time

IF % (In Full Delivery) – Percentage of orders delivered in full quantity

LIFR % (Line Fill Rate) – Order line fulfillment performance

VOFR % (Volume Fill Rate) – Quantity fulfillment performance

ADDD (Average Days of Delayed Delivery) – Average delay in delivery

# 📊 Dashboard Features
## 1️⃣ KPI Toggle Buttons

Users can switch between OTIF, OT, IF, LIFR, and VOFR performance trends using interactive buttons.
This is implemented using Power BI Bookmarks and Button Actions.

## 2️⃣ Performance Trend Analysis

Line charts display KPI performance across time periods to track improvements or declines in delivery performance.

## 3️⃣ Delay Distribution Analysis

Bar charts show how many orders were delivered:

Early

On time

1 day late

2 days late

3 days late

## 4️⃣ Customer Performance Insights

Tables highlight customer-level supply chain performance to identify customers with lower service levels.

## 5️⃣ Interactive Dashboard

Users can interact with:

KPI switching buttons

Visual filtering

Dynamic charts
